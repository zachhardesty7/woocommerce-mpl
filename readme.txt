== Description ==

This repository serves to host a custom WordPress extension that extends WooCommerce. The tweaks are designed specifically for https://MarkPorterLive.com but can be applied anywhere. Credit is given for sources of inspiration. Developed by Zach Hardesty - http://zachhardesty.com.

== Installation ==

1. Upload the entire 'woocommerce-mpl' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
